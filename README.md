# Modern Classics

Classic texts, retold in contemporary English.

Takes old public domain books with archaic language and retells them in a
modern, conversational voice — faithful to the original meaning but genuinely
fun to read.

## How It Works

Each book is translated chapter-by-chapter using Claude, guided by a
text analysis that establishes consistent vocabulary mappings and
a translation strategy for the whole work.

The translator persona is an expert scholar who has studied the work
their entire life, retelling it for a modern audience. Target reading
level: eighth grade to high school.
